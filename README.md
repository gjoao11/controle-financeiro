# Controle financeiro

## Requisitos

- Git
- Node.js
- Yarn ou NPM

## Iniciando

```bash
  # Clone o repositório
  $ https://github.com/gjoao11/controle-financeiro.git

  # Acesse o diretório
  $ cd controle-financeiro

  # Instale as dependências
  $ yarn

  # Acesse `http://localhost:3000` no browser
```
